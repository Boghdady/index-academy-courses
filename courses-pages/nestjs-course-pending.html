ماذ سوف تتعلم في هذه الدورة

  • هتعرف ايه هي ( Nest.js Framework ) وإيه الفوائد اللي وفرتها لينا

  • هنتعرف علي ال ( Building Blocks ) اللي ( Nest.js ) وفرتها لكل جزء من اجزاء ال ( Request Response Cycle )

  • هتتعلم ال ( Validation ) شغال ازاي داخل ( Nest.js ) عشان تقدر تعمل ( Validate ) لل ( Data )

  • هتتعلم ال ( Nest Built-in Interceptor ) وكمان ازاي تعمل ال ( Custom Interceptor ) وال ( Use Cases ) اللي بتستخدم فيها ال (Interceptor)

  • هنشوف ال ( Dependency Injection ) شغال ازاي داخل ( Nest.js )

  • هتتعلم ال ( Nest Providers ) بالتفصيل وهتتعرف علي أنواعهم وال Use Cases اللي بنستخدمها فيه مع تطبيق مثال عملي

  • هتتعرف علي ال ( Scopes ) الخاصة ب ( Nest Building Blocks )

  • هتتعلم عن ال ( Exception Filters ) وازاي ال (Layer) دي بتتعامل مع ال ( Exceptions ) اللي بتحصل داخل التطبيق

  • هتتعلم عن ال ( Guards ) وال ( Middlewares ) وال ( Use Cases ) اللي بتستخدمها فيهم

  • هتتعلم عن ال ( App Configurations ) وهنشوف ازاي ( Nest ) بتوفر ( Config Service ) جاهزة نقدر نستخدمها عشان نوصل لل ( App Configurations )

  • هنتعلم عن ال ( Logging ) وهنشوف ازاي ( Nest ) بتوفر ال( Logging Service ) جاهزة نقدر نستخدمها

  • هنربط ( Nest App ) بالداتا بيز سواء كان (Relational Database like PostgresSQL ) او Non-Relational Database Like MongoDB بمثال عملي
